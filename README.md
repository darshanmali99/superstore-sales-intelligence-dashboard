# Superstore Sales Intelligence Dashboard

An interactive Power BI dashboard analyzing sales performance, profitability, and discount impact using the Superstore dataset.

## 🔍 Key Insights
- Tables & Bookcases generate losses despite strong sales
- Discounts above 30% consistently reduce profitability
- West region leads total sales performance
- Consumer segment contributes over 50% of revenue

## 📊 Dashboard Pages
**Page 1 — Executive Overview**
- KPIs & monthly sales trend
- Category performance
- Loss-making sub-categories

**Page 2 — Regional & Segment Performance**
- Geographic sales distribution
- Regional profit comparison
- Segment contribution

**Page 3 — Discount Impact Analysis**
- Profit vs discount relationship
- Loss-making orders percentage
- Discount threshold insights

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- PostgreSQL (SQL analysis)
